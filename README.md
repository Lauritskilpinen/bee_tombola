# Biavler Tombola

A cheerful Angular app that simulates a tombola lottery machine with a bee-themed animated draw flow.

## Features

- Draws unique tickets from `1` to `1000`
- Random ticket color assignment (`red` or `blue`)
- Animated tombola spinning and bee movement during each draw
- Prominent latest-ticket reveal
- Live results table with draw order, number, and color
- Reset support to start the lottery over
- Responsive two-column layout (machine left, results right)

## Run Locally

1. Install dependencies:

	```bash
	npm install
	```

2. Start development server:

	```bash
	npm start
	```

3. Build for production:

	```bash
	npm run build
	```
